# Bechtel AI Enablement Candidate Vision

An independent candidate campaign by Russell Dudek for the AI Enablement Lead role in Bechtel Manufacturing & Technology.

## Candidate thesis
Build AI like work that must survive the project. The campaign presents an AI Capability Work Package operating model that connects workflow, business consequence, human authority, data and evidence, ownership, delivery pattern, value receipt, and reuse.
